Project Introduction

Our Home Security Application, HomeDefender, is designed to provide enhanced security for your home without the need for costly security systems. Our application integrates various security features, including real-time alerts, to ensure the safety and security of homeowners. This comprehensive and cost-effective solution ensures your home is always protected, giving you peace of mind no matter where you are.

  1. Real-Time Camera Live Feed
    • Integrated a live camera feed displayed via a responsive web interface, allowing users to monitor activity in real time.
  2. Multi-Factor Access Control
    • Supports RFID card authentication and keypad-based PIN entry for secure door access.
    • OTP generation and verification integrated through the dashboard and Telegram bot for added security.
  3. Motion Detection and Alarm Activation
    • Utilizes motion sensors to detect unauthorized activity and trigger alarms.
    • Sends notifications for motion events, with optional Telegram alerts.
  4. Dashboard for Centralized Control
    • Intuitive Flask-based web dashboard featuring buttons to lock/unlock doors, activate/deactivate motion sensors, and generate OTPs.
    • Displays activity logs and provides a link to view the live camera stream.
  5. Dynamic MQTT Communication
    • Implements MQTT protocol for seamless communication between devices and sensors.
    • Publishes and subscribes to topics for real-time status updates and controls.
  6. Embedded System Integration
    • Uses Raspberry Pi GPIO pins for connecting sensors, LEDs, buzzer, servo motor, and RFID reader.
    • Integrates with PiCamera for video streaming.
  7. Responsive Web Design
    • Designed with Bootstrap and custom CSS for a modern and responsive user interface.
  8. Database and Notification Support
    • Connected to a MySQL database for managing RFID, PIN, and OTP records.
    • Optional Telegram bot integration for immediate security updates.
  9. Enhanced Security Features
    • Auto-locks the door after access, with an alarm system for unauthorized attempts.
    • Handles invalid access gracefully with notifications and visual alerts.
  10. Extensible and Modular Codebase
    • Modular Python implementation with dedicated classes for client and hub functionalities.
    • Extensible architecture for future feature additions like facial recognition or cloud integration.
