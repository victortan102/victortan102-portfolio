Project Introduction

This project implements a robotic car control system based on the Raspberry Pi Pico W using FreeRTOS. The car is equipped with various sensors, such as the HC-SR04 Ultrasonic Distance Sensor, WYC-H206 Wheel Encoder Module, and the IR Line Tracking Module. It integrates motor control, ultrasonic distance measurement, WiFi communication, barcode decoding, and line-following capabilities to execute tasks such as remote navigation, obstacle avoidance, and real-time dashboard updates. Additionally, the car can be controlled remotely using a GY-511 accelerometer module, which acts as a motion-based remote control connected to another Raspberry Pi Pico.

The Autonomous Robotic Car, powered by a Raspberry Pi Pico W and FreeRTOS, boasts the following key features, enhancing its performance and user experience:

  1. Precision Motor Control
    • Real-time PID control for maintaining stable speeds of both wheels, ensuring smooth navigation.
    • Supports precise 90-degree turns using a combination of PID and wheel encoder feedback for accurate movement.
  2. Ultrasonic Obstacle Detection
    • Ultrasonic sensors with Kalman filtering for accurate distance measurement.
    • Automatic emergency braking when obstacles are detected within a specified range.
  3. Wireless Connectivity and Control
    • Wi-Fi-enabled communication for remote control via UDP commands.
    • Support for commands like forward, reverse, stop, and directional turns with real-time feedback.
  4. Dynamic Task Management
    • Multi-tasking capabilities using FreeRTOS for modular and efficient operation.
    • Dedicated tasks for motor control, obstacle detection, line-following, and communication.
  5. Line Following and Barcode Scanning
    • Infrared (IR) sensors for line detection and navigation.
    • Advanced barcode recognition for unique identification using predefined patterns.
  6. Dashboard Integration
    • Real-time telemetry and feedback sent to a remote dashboard.
    • Displays data like speed, distance travelled, and obstacle proximity for enhanced monitoring.
  7. Adaptive Speed and Direction
    • Real-time adjustments to speed and direction using accelerometer-based tilt control.
    • Comprehensive handling of terrain and directional changes.
  8. Robust Software Architecture
    • FreeRTOS integration for real-time performance with flexible task priority.
    • Modular code structure for easy maintenance and feature expansion.
  9. Energy Efficiency
    • Optimized power consumption through controlled motor usage and sensor activation.
  10. Scalable Design
    • Easy to expand with additional sensors or modules for advanced functionalities like GPS integration or camera-based navigation.

These features make the car a versatile platform suitable for applications ranging from education and research to practical robotics solutions.
