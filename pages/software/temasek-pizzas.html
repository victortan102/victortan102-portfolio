Project Information

  • Category: Software, Object-Oriented Programming
  • Language: Java
  • Project Date: April to August 2019

Project Description

The project's objective is to develop a Java application that enables the user to login into the application as a form of authentication to place orders of pizzas and drinks. The application must also let the customer edit, delete, and place multiple orders during the program run-time.

When the user closes the program, a message box with a summary of all the orders will also be shown to the user.

The company Temasek Pizzas also hopes that we can include images in the menu to increase the user-friendliness of the application while helping customers view the menu better online.
