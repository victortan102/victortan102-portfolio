Project Overview

The PacificMeta Game Analysis Dashboard focuses on the design and implementation of an AI-powered analytics system for healthcare training games. Developed in collaboration with Saint Francis University and Pacific Meta, the project aims to enhance the educational value of serious games by transforming raw gameplay telemetry such as user decisions, errors, and timing into actionable insights.
