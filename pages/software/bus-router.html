Project Description

Our application - BusRouter, is a smart routing system designed for an intended reduction in costs for a tourist bus operator in Singapore.

The application also offers a way to assist bus operators in planning their routes by generating the best route based on their requirements.

This entails applying Dijkstra's algorithm on each node that results in the formation of a distance dictionary which in turn is processed by Ant Colony Optimization or Held-Karp algorithm, depending on the number of destinations. The system optimizes routes based on three key criteria, including length/distance, time and the minimum ERP that is required with optional consideration of current traffic.

  1. Dynamic Route Optimization:

    The application calculates optimal routes based on user preferences such as shortest distance, fastest time, or avoiding ERP (Electronic Road Pricing) charges.

  2. Interactive User Interface:

    An intuitive UI allows users to select pickup points, hotel drop-off locations, and filter routes by star ratings of hotels. It also includes a map view for visualizing routes.

  3. Customizable Settings:

    Users can enable or disable features like traffic consideration and database updates, and upload ERP gantry data for accurate route optimization.

  4. Real-Time Features:

    The map embed reloads dynamically to ensure up-to-date route data is displayed.

  5. Dark Mode Support:

    A built-in toggle allows users to switch between light and dark modes for better visibility and user comfort.

  6. Hotel Star Rating Integration:

    The application integrates with a hotel database, dynamically fetching and displaying hotels based on user-specified star ratings.

  7. Database Integration:

    Seamlessly fetches and updates data in a MongoDB database to maintain up-to-date information on hotels and routes.

  8. Automated Scraping:

    A backend scraper fetches hotel star ratings from Google, ensuring the database is consistently updated with accurate information.

  9. Progress Tracking:

    Displays a progress bar during route calculation, enhancing user experience by providing real-time feedback.

  10. Modal Confirmation:

    Ensures users can review their selections before submission through a confirmation dialog.

This combination of features highlights the BusRouter application's focus on user convenience, customization, and real-time data integration.
