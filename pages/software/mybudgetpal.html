Project Overview

MyBudgetPal is a secure, cloud-based budget tracking web application designed to help individuals manage their personal finances effectively. The app allows for manual entry of personal expenses, as well as automatic scanning of receipts using OCR technology. The user can also view their spending data through an integrated dashboard. It also features a friend-sharing system, allowing expenses to be shared, split, and tracked together. This application is designed to be both secure and user-friendly, hosted on AWS and developed using web technologies such as Flask.
