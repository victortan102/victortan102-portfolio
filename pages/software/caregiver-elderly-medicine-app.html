Project Description

This application is designed to help caregivers and elderly users manage and track daily medication schedules. The app provides reminders, tracking, and essential functionalities to support caregivers in assisting elderly individuals with medication adherence.

This Android application is a front-end prototype developed to explore HCI principles, emphasizing usability and design, without incorporating backend functionality.

  3. Medication Tracking: Elderly users can mark medications as taken, and caregivers can check adherence.
  4. Profile Management: Supports editing profiles and setting preferences, including language and font size adjustments.
  5. Dark Mode and Language Support: Provides options for dark mode and multiple language selections (English, Chinese, Malay).
