Project Overview

This application is a comprehensive GUI-based steganography tool designed to hide data (text or files) within various media formats, including images, audio, and video. It leverages AES-256 encryption and key-based pseudo-random embedding patterns to ensure secure and stealthy data transmission.

ðŸŽ¯ Key Features of Application

  1. Multi-Modal Support: Supports .png, .bmp images, .wav audio, and .mp4 video files.
  2. Security & Encryption: Optional AES-256 CBC encryption for payloads using PBKDF2 for key derivation.
  3. Key-Based Permutation: Uses a user-provided key to derive a pseudo-random embedding plan, including start position, pixel/sample selection sequence, and LSB patterns.
  4. Integrity Verification: Employs 16-bit checksums and magic number headers to verify data integrity upon extraction.

  5. Advanced Video Steganography:

    • Frame-based: Embeds data into specific frames (e.g., every Nth frame) and rebuilds the video using the lossless FFV1 codec to prevent data loss.
    • Audio-based: Extracts the audio stream from an MP4, embeds data, and muxes it back using FFmpeg.

  6. Analytical Visualizations: Features comparison tools for Image (difference maps) and Audio (waveforms, spectrograms, and histograms) to analyze embedding impact.
